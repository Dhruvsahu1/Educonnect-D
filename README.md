# 🚀 Overview



EduConnect is a cloud-native academic and social networking platform designed to bridge the gap between students and educational institutions.



Students can showcase certifications, engage in discussions, collaborate academically, and access study resources, while administrators manage institutions, moderate content, and distribute educational materials.



The platform is engineered using modern DevSecOps practices and deployed on Kubernetes with security embedded throughout the software delivery lifecycle.
